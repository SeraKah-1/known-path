# Demo script (~3 minutes)

1. **Problem (15s)** — Agents pick the wrong table when names look similar (`revenue_old` vs canonical).
2. **Baseline (30s)** — `kp run --mode baseline` — many fetches, trap lit.
3. **Known path (45s)** — `kp run --mode known-path` — two green nodes, SQL artifact.
4. **Fail closed (30s)** — `kp run --mode blocked` — stop, no invented table.
5. **Leave the route (20s)** — show `examples/runs/writeback_route_note.md`.
6. **Close (20s)** — metrics table from `kp demo`; link DataHub hackathon + repo.
